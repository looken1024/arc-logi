# 表情包小程序

一个基于H5的表情包小程序，适配所有手机型号，支持表情包浏览、搜索、收藏和分享等功能。

## 功能特性

### 🎯 核心功能
- **表情包浏览** - 网格式布局，支持大量表情包展示
- **分类筛选** - 热门、搞笑、萌系、表情、梗图等分类
- **搜索功能** - 实时搜索表情包标题
- **预览查看** - 点击表情包可查看大图
- **收藏管理** - 本地收藏喜欢的表情包
- **下载分享** - 支持下载和分享表情包
- **随机表情** - 随机推荐表情包

### 📱 移动端优化
- **响应式设计** - 完美适配所有手机屏幕尺寸
- **触摸优化** - 针对触摸操作优化交互体验
- **横屏适配** - 支持横屏浏览
- **暗黑模式** - 自动适配系统暗黑主题

### 🎨 用户体验
- **流畅动画** - 优雅的过渡效果和微交互
- **懒加载** - 图片懒加载，提升性能
- **无限滚动** - 滚动到底部自动加载更多
- **离线收藏** - 收藏数据本地存储

## 技术实现

### 前端技术
- **HTML5** - 语义化标签，支持移动端特性
- **CSS3** - Flexbox/Grid布局，响应式设计
- **JavaScript ES6+** - 模块化开发，现代语法

### 移动端适配
- `viewport` 设置确保移动端正确缩放
- `rem/vw/vh` 单位实现响应式布局
- CSS Grid 和 Flexbox 弹性布局
- 媒体查询适配不同屏幕尺寸

### 性能优化
- 图片懒加载 (`loading="lazy"`)
- 防抖搜索减少请求
- 本地存储减少网络请求
- CSS动画使用GPU加速

## 配置说明

### 修改表情包数据
在 `script.js` 中的 `generateEmojiData()` 方法中修改表情包数据源：

```javascript
// 替换为实际的图片URL
url: `https://your-domain.com/images/emoji${i}.jpg`
```

### 自定义分类
修改 `categories` 数组添加新分类：

```javascript
const categories = ['popular', 'funny', 'cute', 'emotion', 'meme', 'your-category'];
```

### 调整页面大小
修改 `pageSize` 变量控制每页显示数量：

```javascript
this.pageSize = 20; // 每页显示20个表情包
```

## 部署建议

### 静态部署
- 可部署到 GitHub Pages、Vercel、Netlify 等
- 确保HTTPS访问以支持现代Web特性

### 服务器部署
- 建议配置Gzip压缩
- 设置合适的缓存策略
- 启用CDN加速图片访问
