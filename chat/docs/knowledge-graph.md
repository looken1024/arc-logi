# 知识图谱可视化组件文档

## 概述

知识图谱可视化组件用于以图形方式展示知识库中不同知识条目之间的关联关系。该组件使用 vis.js 网络图库实现，支持节点和边的可视化、交互操作、搜索筛选和导出功能。

## 功能特性

### 1. 可视化展示
- **节点（知识条目）**：以圆点形式展示，不同类型使用不同颜色区分
  - 文本：绿色 (#10a37f)
  - 问答：蓝色 (#3b82f6)
  - 概念：橙色 (#f59e0b)
  - 流程：红色 (#ef4444)
  
- **边（关系）**：以带箭头的线条展示，不同关系类型使用不同颜色
  - 相关：紫色 (#8b5cf6)
  - 包含：粉色 (#ec4899)
  - 从属：青色 (#14b8a6)
  - 相似：橙色 (#f97316)
  - 标签：天蓝色 (#06b6d4)

### 2. 交互功能
- **缩放**：支持鼠标滚轮缩放和按钮缩放
- **拖拽**：支持拖拽节点调整位置
- **节点点击**：单击节点显示详情（类型、标题、内容预览）
- **节点聚焦**：双击节点放大并聚焦该节点
- **适应屏幕**：一键将图谱调整为适合屏幕的大小

### 3. 搜索和筛选
- **节点搜索**：通过标题或内容搜索节点，高亮匹配项
- **关系筛选**：按关系类型筛选显示的边

### 4. 导出功能
- **PNG 导出**：导出为 PNG 图片格式
- **SVG 导出**：导出为 SVG 矢量图形格式

### 5. 性能优化
- 支持大规模知识图谱展示（默认最多 500 个节点）
- 使用物理引擎自动布局
- 支持懒加载和分页

## API 接口

### 获取图谱数据

```
GET /api/knowledge-bases/<kb_id>/graph
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| kb_id | int | 是 | 知识库 ID |
| search | string | 否 | 搜索关键词 |
| types | string | 否 | 关系类型筛选，多个用逗号分隔 |
| limit | int | 否 | 节点数量限制，默认 500 |
| offset | int | 否 | 偏移量，默认 0 |

**响应示例：**

```json
{
  "nodes": [
    {
      "id": "item_123",
      "label": "知识条目标题",
      "type": "text",
      "title": "知识条目标题",
      "content": "内容预览..."
    }
  ],
  "edges": [
    {
      "id": "relation_1",
      "from": "item_123",
      "to": "item_456",
      "relationType": "related",
      "label": "相关"
    }
  ],
  "statistics": {
    "totalNodes": 50,
    "totalEdges": 100,
    "displayedNodes": 50,
    "displayedEdges": 100,
    "relationTypes": {
      "related": 30,
      "parent": 10,
      "child": 10,
      "similar": 25,
      "tag": 25
    }
  }
}
```

### 导出图谱

```
POST /api/knowledge-bases/<kb_id>/graph/export
```

**请求体：**

```json
{
  "format": "png"  // 或 "svg"
}
```

**响应示例：**

```json
{
  "success": true,
  "format": "png",
  "data": {
    "nodes": [...],
    "edges": [...]
  }
}
```

## 使用说明

### 访问图谱视图

1. 打开知识库管理页面
2. 选择一个知识库
3. 点击顶部标签栏的"关系图谱"标签

### 基本操作

1. **查看节点详情**：单击任意节点，可看到节点类型、标题和内容预览
2. **聚焦节点**：双击任意节点，图谱会放大并聚焦到该节点
3. **调整视图**：
   - 滚动鼠标滚轮进行缩放
   - 点击放大/缩小按钮进行调整
   - 点击"适应屏幕"按钮将图谱调整为最佳显示大小
4. **拖拽节点**：按住节点拖拽可以调整其在图谱中的位置

### 搜索和筛选

1. **搜索节点**：
   - 在图谱工具栏的搜索框中输入关键词
   - 系统会自动筛选并显示匹配的节点
   
2. **筛选关系**：
   - 点击"筛选关系类型"下拉框
   - 选择要显示的关系类型
   - 可以多选多个类型

### 导出图谱

1. **导出为 PNG**：
   - 点击工具栏的"PNG"按钮
   - 图片会自动下载到本地
   
2. **导出为 SVG**：
   - 点击工具栏的"SVG"按钮
   - 矢量图形文件会自动下载到本地

### 图例说明

图谱左下角显示图例，说明各种关系类型对应的颜色：
- 相关：紫色连线
- 包含：粉色连线
- 从属：青色连线
- 相似：橙色连线
- 标签：天蓝色连线

## 性能考虑

1. **节点数量**：默认限制为 500 个节点，大型知识库建议使用搜索和筛选功能
2. **物理模拟**：首次加载时会进行物理模拟布局，节点会自动分布
3. **响应式设计**：图谱容器会自动适应不同屏幕尺寸
4. **懒加载**：支持分页加载大量节点

## 技术实现

- **可视化库**：vis.js (vis-network)
- **前端框架**：原生 HTML/CSS/JavaScript
- **后端框架**：Flask
- **数据库**：MySQL
- **CDN 引入**：
  - vis-network CSS: `https://unpkg.com/vis-network@9.1.6/standalone/umd/vis-network.min.css`
  - vis-network JS: `https://unpkg.com/vis-network@9.1.6/standalone/umd/vis-network.min.js`

## 常见问题

### Q: 为什么图谱加载很慢？
A: 可能是因为节点数量过多，建议使用搜索或筛选功能减少显示的节点数量。

### Q: 如何调整节点大小？
A: 当前版本不支持调整节点大小，后续版本可能会添加此功能。

### Q: 导出的图片是空白怎么办？
A: 确保浏览器允许弹出窗口，有些浏览器需要手动允许才能下载文件。

### Q: 节点重叠看不清怎么办？
A: 可以使用缩放功能放大视图，或者拖拽节点调整位置使其分散开。
