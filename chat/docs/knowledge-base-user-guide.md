## 知识库管理

### 创建知识库

1. 点击右上角「新建知识库」按钮
2. 填写知识库名称和描述
3. 点击「保存」

**示例**:
```
名称: Python 技术文档
描述: 收集整理 Python 相关技术知识
```

### 编辑知识库

1. 在知识库列表中，点击右侧「编辑」图标
2. 修改名称或描述
3. 点击「保存」

### 删除知识库

1. 在知识库列表中，点击右侧「删除」图标
2. 确认删除提示
3. 知识库及其所有内容将被永久删除

**注意**: 删除操作不可恢复，请谨慎操作。

## 知识条目操作

### 添加知识条目

1. 选择要添加条目的知识库
2. 点击「添加条目」按钮
3. 填写表单:
   - **标题**: 条目的名称
   - **类型**: 选择知识类型
   - **内容**: 填写知识详情

**知识类型说明**:

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 文本 | 普通文本内容 | 文档、说明、指南 |
| 问答 | 问题与答案 | FAQ、技术问答 |
| 概念 | 概念定义 | 术语解释、定义 |
| 流程 | 操作步骤 | 操作手册、流程说明 |

### 编辑知识条目

1. 在条目列表中，点击「编辑」图标
2. 修改内容
3. 点击「保存」

### 删除知识条目

1. 在条目列表中，点击「删除」图标
2. 确认删除

## 搜索与筛选

### 快速搜索

1. 在页面顶部的搜索框中输入关键词
2. 系统实时显示搜索结果

### 高级筛选

在知识库详情页面，可以按以下条件筛选:

- **类型**: 文本/问答/概念/流程
- **标签**: 选择特定标签
- **时间**: 创建时间范围

### 搜索技巧

1. **精确匹配**: 使用双引号 `"关键词"`
2. **或搜索**: `关键词1 OR 关键词2`
3. **排除**: `关键词1 NOT 关键词2`

## 知识图谱

### 查看知识图谱

1. 在知识库详情页面，点击「关系图谱」标签
2. 系统加载并显示知识网络图

### 图谱交互

| 操作 | 方法 |
|------|------|
| 移动视图 | 拖拽空白区域 |
| 缩放 | 鼠标滚轮 |
| 选中节点 | 点击节点 |
| 查看详情 | 双击节点 |

### 节点类型

| 颜色 | 类型 |
|------|------|
| 蓝色 | 文本 |
| 绿色 | 问答 |
| 黄色 | 概念 |
| 红色 | 流程 |
| 灰色 | 标签 |

### 导出图谱

1. 点击「PNG」或「SVG」按钮
2. 选择保存位置

## 版本管理

### 查看历史版本

1. 点击知识条目右侧的「历史」图标
2. 查看版本列表

**版本信息包括**:
- 版本号
- 修改内容
- 修改时间
- 修改人

### 回滚版本

1. 在版本历史中，选择要回滚的版本
2. 点击「回滚」按钮
3. 确认操作

**注意**: 回滚会创建新版本，原内容不会丢失。

## 标签与分类

### 添加标签

创建或编辑知识条目时，在「标签」字段添加:

```
示例标签: python, 编程, 教程
```

### 管理标签

- 标签会自动创建
- 相同名称的标签会合并
- 标签显示在图谱中为独立节点

### 分类管理

可以为知识库添加层级分类:

```
技术文档
├── 编程语言
│   ├── Python
│   └── JavaScript
└── 数据库
    ├── MySQL
    └── Redis
```

## AI 对话集成

### 在聊天中使用知识库

在 AI 对话中，你可以:

1. **查询知识**:
   ```
   用户: 查询知识库中关于 Python 的内容
   ```

2. **写入知识**:
   ```
   用户: 将 Python 虚拟环境的使用方法写入知识库
   ```

3. **分析知识**:
   ```
   用户: 分析知识库的内容趋势
   ```

### 对话命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/kb search <关键词>` | 搜索知识 | `/kb search python` |
| `/kb write <内容>` | 写入知识 | `/kb write Python 基础教程...` |
| `/kb list` | 列出知识库 | `/kb list` |
| `/kb stats` | 统计信息 | `/kb stats` |

### 智能推荐

系统会根据对话上下文自动推荐相关知识:

- 当你询问某个主题时
- 系统会提示相关的知识条目
- 点击即可查看完整内容

## 常见问题

### Q: 如何导入大量知识？

A: 可以通过 API 批量导入:

```bash
curl -X POST /api/knowledge-bases/1/items/batch \
  -H "Content-Type: application/json" \
  -d '{"items": [...]}'
```

### Q: 知识库有容量限制吗？

A: 单知识库最大支持 10,000 条记录。

### Q: 如何分享知识库？

A: 目前支持:
1. 导出为 JSON/CSV
2. 复制知识条目链接
3. 团队共享 (企业版)

### Q: 数据丢失怎么办？

A: 系统提供:
1. 版本历史回滚
2. 操作历史记录
3. 定期自动备份
