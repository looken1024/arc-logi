# Random Joke Skill

随机笑话技能，用于在对话中提供轻松幽默的笑话。

## 功能描述

此技能用于获取一个随机笑话，为对话增添趣味性。当用户想要放松心情、缓解紧张气氛或单纯想听个笑话时，AI 可以调用此技能。

## 使用场景

- 用户请求："讲个笑话吧"
- 用户心情低落时："说点开心的"
- 聊天冷场时："活跃一下气氛"
- 儿童用户请求："给我讲个笑话"

## 参数说明

此技能无需参数，直接调用即可。

## 返回值

返回 JSON 对象，包含以下字段：

```json
{
  "joke": "笑话内容",
  "category": "笑话分类",
  "length": "笑话字数",
  "language": "zh-CN"
}
```

## 使用示例

### 示例 1: 获取随机笑话

**用户**: 讲个笑话吧

**AI 调用**:
```json
{
  "function": "random_joke",
  "arguments": {}
}
```

**返回**:
```json
{
  "joke": "为什么程序员喜欢用黑色背景？因为亮色背景会吸引虫子！",
  "category": "程序员笑话",
  "length": 20,
  "language": "zh-CN"
}
```

**AI 回复**: 为什么程序员喜欢用黑色背景？因为亮色背景会吸引虫子！

### 示例 2: 请求另一个笑话

**用户**: 再来一个

**AI 调用**:
```json
{
  "function": "random_joke",
  "arguments": {}
}
```

**返回**:
```json
{
  "joke": "程序员去餐厅点餐：\"我要一份炒饭，不要放盐。\" 服务员：\"好的，不放盐。\" 程序员：\"等等，还是放盐吧，我刚刚说的是默认值。\"",
  "category": "程序员笑话",
  "length": 35,
  "language": "zh-CN"
}
```

**AI 回复**: 程序员去餐厅点餐："我要一份炒饭，不要放盐。" 服务员："好的，不放盐。" 程序员："等等，还是放盐吧，我刚刚说的是默认值。"

## 笑话库

技能内置以下笑话类别：

1. **程序员笑话** - 与编程、开发相关的幽默
2. **日常笑话** - 日常生活中有趣的场景
3. **儿童笑话** - 适合儿童理解的简单笑话
4. **冷知识笑话** - 结合冷知识的幽默
5. **谐音梗笑话** - 中文谐音带来的趣味

每次调用会从笑话库中随机选择一个笑话返回。

## 技术实现

- **主脚本**: `skill.py` - 技能核心实现
- **文档**: `SKILL.md` - 详细使用文档
- **依赖**: Python 标准库 (`random`)
- **语言**: 主要支持中文笑话
- **随机性**: 使用系统随机数生成器确保随机性

## 错误处理

如果执行过程中出现错误，返回格式如下：

```json
{
  "error": "获取笑话失败: [错误详情]"
}
```
