# 联网新闻搜索技能 (web_search_news)

## 概述

联网新闻搜索技能允许AI助手通过联网搜索获取最新新闻和热点事件。当用户询问关于最新消息、新闻事件、时事信息等问题时，此技能会自动联网搜索并返回相关新闻内容。

## 功能特点

- 联网搜索最新新闻和事件
- 支持多种新闻分类（综合、科技、商业、体育、娱乐、科学、健康）
- 支持中英文搜索
- 返回新闻标题、链接和摘要

## 使用场景

1. 用户询问"最近有什么重大新闻？"
2. 用户询问"今天的科技新闻有哪些？"
3. 用户询问"最近发生了什么重大事件？"
4. 用户询问"最新的体育赛事新闻"
5. 用户询问"关于XXX的最新消息"

## 参数说明

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| query | string | 是 | - | 搜索关键词或新闻主题 |
| num_results | integer | 否 | 5 | 返回的新闻数量 |
| category | string | 否 | general | 新闻分类：general/tech/business/sports/entertainment/science/health |
| language | string | 否 | zh | 搜索语言：zh(中文)/en(英文) |

## 返回结果示例

```json
{
  "success": true,
  "query": "人工智能",
  "category": "tech",
  "language": "zh",
  "count": 5,
  "search_time": "2026-02-21 10:30:00",
  "results": [
    {
      "title": "AI领域重大突破",
      "url": "https://example.com/news/1",
      "snippet": "最新的人工智能研究取得重大突破..."
    }
  ]
}
```

## 注意事项

- 此技能需要联网才能使用
- 搜索结果来自第三方搜索引擎
- 返回的链接可能包含广告或无关内容，请注意筛选
