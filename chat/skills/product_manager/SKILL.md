# 产品经理技能

产品经理技能，用于提出有建设性的产品方案和创意。

## 功能描述

此技能用于生成针对特定领域、问题或创意的产品方案。当用户需要产品创意、功能规划、市场定位建议时，AI 可以调用此技能。

## 使用场景

- 用户请求："帮我设计一个社交 App"
- 用户需要创意："有什么好的产品点子？"
- 产品优化："如何改进电商购物体验？"
- 市场分析："针对年轻人的新产品方向"

## 参数说明

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| topic | string | 是 | 产品主题、领域或问题描述，例如："社交应用"、"电商平台"、"健康管理" |
| detail_level | string | 否 | 方案详细程度：brief（简要）、normal（正常）、detailed（详细），默认 normal |
| target_audience | string | 否 | 目标用户群体，例如："年轻人"、"上班族"、"老年人" |
| innovation_level | string | 否 | 创新程度：incremental（渐进式改进）、breakthrough（突破性创新）、radical（颠覆性创新），默认 incremental |

## 返回值

返回 JSON 对象，包含以下字段：

```json
{
  "success": true,
  "topic": "原始主题",
  "target_audience": "目标用户",
  "product_name": "产品名称",
  "product_concept": "产品核心理念",
  "key_features": ["功能1", "功能2", ...],
  "market_positioning": "市场定位描述",
  "innovation_points": ["创新点1", "创新点2", ...],
  "implementation_considerations": "实施注意事项",
  "next_steps": ["下一步1", "下一步2", ...]
}
```

## 使用示例

### 示例 1: 生成社交应用方案

**用户**: 帮我设计一个面向年轻人的社交应用

```json
  }
}
```

## 方案生成逻辑

1. **需求分析**: 根据topic和target_audience分析用户需求和痛点
2. **概念生成**: 结合innovation_level生成产品核心理念和名称
3. **功能规划**: 基于detail_level设计关键功能和创新点
4. **市场定位**: 确定目标市场和差异化优势
5. **实施建议**: 提供技术实现、推广策略和下一步行动

## 技术实现

- **主脚本**: `skill.py` - 技能核心实现
- **文档**: `SKILL.md` - 详细使用文档
- **依赖**: OpenAI Python库（可选，用于高质量方案生成）
- **语言**: 主要支持中文，可根据需要支持其他语言
- **AI集成**: 可调用GPT模型生成创新性方案，也可使用预定义模板

## 错误处理

如果执行过程中出现错误，返回格式如下：

```json
{
  "success": false,
  "error": "生成产品方案失败: [错误详情]"
}
```
